# Prettij Markup Language

**Prettij** is a straight-forward, unambiguous markup language designed to look as good on screen as it parses under the hood.

### Examples

##### Assignments

```
key = value
```

##### Data types

```
text = "hello"
number = 2025
bool = true
undefined = null
float = 3.14
```

##### Arrays

```
array = ["first", "second", "third"]
```

#### Objects

```
object = {
    key1 = value1,
    key2 = value2,
}
```
